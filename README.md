# README

Configuration:

ruby: 3.1.2

rails: 7.0.4

How to Run Locally:
- Download repo
- Run "bundle config set --local without 'production'"
- Run "bundle"
- Run "rails server"
- navigate to 'http://localhost:3000/' on any browser
